# To-Do
Simple To-Do terminal app, i could have added Deleting tasks or save file but too lazy to do it without money :p...

I made this project to get more comfortable with C# and practice solving problems on my own instead of following tutorials.

Right now, the app lets you add tasks and view them through a simple console menu. The tasks are stored while the program is running.

Building this project helped me understand things like loops, `if` statements, lists, and how to organize a program so different features work together.

I'm planning to keep improving it by adding features like deleting tasks, saving them to a file, loading them when the program starts, and improving the overall user experience.

This project is still a work in progress, and I'll continue updating it as I learn more.
